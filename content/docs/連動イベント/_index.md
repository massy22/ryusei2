# Ｗスロット連動イベント

> [!NOTE]
> **パーフェクトコレクション版をお使いの方へ**  
> パーフェクトコレクションでは、GBAカートリッジ（ロックマンエグゼ）をDS本体に挿入しなくても、Wスロット連動イベントがゲーム内機能として再現されています。  
> 詳しくは [パーフェクトコレクションまとめサイト](https://chipcom.org/ryusei-perfect-collection/game-specs/) をご覧ください。

以下はニンテンドーDS版（流星のロックマン2）の連動イベント情報です。

<br />ゲーム内でキーアイテムが見つかるようになります。<br />
5つの日記を全て入手し、全て読むと熱斗からメールが届き、ウォーロック装備「[エグゼブラスター](docs/ウォーロックの武器)」が手に入ります。
### 対応ソフト
ロックマンエグゼ（アドバンス版）<br />
ロックマンエグゼバトルチップGP
