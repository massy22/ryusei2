# 流星のロックマン2概要
DS「流星のロックマン2」は、「ロックマン」20周年記念の第3弾タイトルとして発売されるシリーズ最新作。「サンダーベルセルク」、「グリーンシノビ」、「ファイアダイナソー」の3種類のロックマンが登場し、「ベルセルク×シノビ」と「ベルセルク×ダイナソー」の2通りの組み合わせで2タイトルが発売される。

## ストーリー
 時は220X年……。
 電波技術の急速な発達により、人々は便利な生活を手に入れていた。

 そんな地球に対し侵略者が現れる。FM星人と呼ばれる宇宙人だ。地球は絶対絶命の危機を迎えたかに思われた。

 しかし、その危機に立ち向かったのは一人の少年と、一人の宇宙人だった。

 少年と宇宙人がひとつに融合することでヒーローが生まれた。

 その名は「流星のロックマン」。

 ロックマンの命がけの戦いにより、地球はFM星人から救われ人々は平和な暮らしを取り戻していった。

 ……それから数ヵ月後。
 物語は「TKシティ」から始まる。 

## 新変身システム「トライブオン」
本作では、かつて地上に存在した強大なチカラを持つ3つの種族（トライブ）の残した遺産「オーパーツ・シノビ（シノビ）」、「オーパーツ・ベルセルク（ベルセルク）」、「オーパーツ・ダイナソー（ダイナソー）」のチカラによって、ロックマン（主人公ほしかわスバル）を変身させることができる。

　3種の遺産には、「シノビ」には木属性が、「ベルセルク」には電気属性が、「ダイナソー」には炎属性というように、それぞれに特徴の異なる属性が付加されている。
